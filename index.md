### 数据收集声明

**本应用不收集任何个人身份信息。**

斗破牙门是一款单机 Roguelike 斗地主卡牌游戏。无需注册或登录账户，不收集姓名、邮箱、电话号码等个人信息，不使用 ATT（App Tracking Transparency）或 IDFA 追踪，不投放任何广告。

### 分析与崩溃报告

为改善游戏体验，本应用使用以下 Firebase 服务：

- **Firebase Analytics**：收集匿名化的使用事件数据（关卡进度、功能使用频率、应用性能等）。这些数据完全匿名，不与您的身份关联。
- **Firebase Crashlytics**：自动收集崩溃报告（崩溃日志、设备型号、操作系统版本）。崩溃报告不包含任何个人信息。

### 本地存储

应用使用 `UserDefaults`、`Keychain` 和 `SwiftData` 在设备本地存储以下信息：

| 存储项 | 用途 | 类型 |
|--------|------|------|
| 音效音量 | 保存玩家音效偏好设置 | 浮点数 |
| 教程完成标记 | 记录玩家是否完成新手教程 | 布尔值 |
| 购买状态 | 记录应用内购买解锁状态（UserDefaults + Keychain） | 布尔值 |
| 游戏进度存档 | 保存当前冒险进度（关卡、分数、手牌等） | 数据库 |

以上数据完全存储于用户设备本地，不会上传至任何服务器。

### 儿童隐私

## English Version

### App Information

- **App Name**: 斗破牙门 (Dou Po Ya Men)
- **Bundle ID**: com.hongzeng.doudizhu
- **Developer**: Individual developer

### Data Collection Statement

**This app does not collect any personally identifiable information.**

Dou Po Ya Men is a single-player Roguelike Doudizhu card game. No account registration or login is required. We do not collect names, emails, phone numbers, or any personal information. We do not use ATT (App Tracking Transparency) or IDFA tracking. We do not serve any advertisements.

### Analytics & Crash Reporting

To improve the gaming experience, this app uses the following Firebase services:

- **Firebase Analytics**: Collects anonymized usage event data (level progress, feature usage frequency, app performance). This data is completely anonymous and not linked to your identity.
- **Firebase Crashlytics**: Automatically collects crash reports (crash logs, device model, OS version). Crash reports do not contain any personal information.

### Local Storage

The app uses `UserDefaults`, `Keychain`, and `SwiftData` to store the following data locally on the device:

| Item | Purpose | Type |
|------|---------|------|
| Sound volume | Save player sound preferences | Float |
| Tutorial completion flag | Track whether player has completed the tutorial | Boolean |
| Purchase status | Track in-app purchase unlock state (UserDefaults + Keychain) | Boolean |
| Game save data | Save current adventure progress (floor, score, hand, etc.) | Database |

All data is stored entirely on the user's device and is never uploaded to any server.

### Third-Party Services

- ✅ **Firebase Analytics**: Anonymous usage statistics ([Firebase Privacy Policy](https://firebase.google.com/support/privacy))
- ✅ **Firebase Crashlytics**: Crash report collection (same as above)
- ✅ **Apple StoreKit**: In-app purchase processing ([Apple Privacy Policy](https://www.apple.com/legal/privacy/))
- ❌ No advertising frameworks
- ❌ No social login
- ❌ No user tracking services

### Children's Privacy

This app does not collect any information from children under 13. Since the app collects no personal data whatsoever, it complies with COPPA requirements.
